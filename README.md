### 使用步骤
1. install requirements
```
pip install -r requirements.txt
```
2. in main.py set the conditions you want.
```python
if __name__ == '__main__':
    results = []
    max_stops = 1
    origins = ['TYO', 'PVG']
    destinations = ['NYC', 'LAX']
    dates = [
        '2023-03-05',
        '2023-03-06',
        '2023-03-07',
        '2023-03-08',
        '2023-03-09',
        '2023-03-10'
    ]
```
3.Run main.py and you will see the output file.
